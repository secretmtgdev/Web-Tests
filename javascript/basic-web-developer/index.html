JavaScript

this keyword

  • Arrow function: defined by the lexical environment of the arrow function
  • Function call on object: refers to the current object, in other cases the window object is referenced

Prototypes

  • A template object in which other objects may stem from. When an object's parent has a prototype this is referred to as a prototype chain
  • Properties and methods are bound to the prototype and not the object instance
  • Properties and methods are inherited if and only if they follow the pattern of Object.prototype
  • Every prototype has a constructor which points to the original constructor.

Classes

  • Utilizes prototype inheritance
  • Body of class is bound by strict mode(turns former mistakes in errors)

Immediate Invoked Functions

Closures

  • Lexical scoping: inner function having access to variables of the outer function
  • Closure: Variable assigned to a functions inner function which maintains lexical scoping
  • Have access to Local, Global, and outer function scoping
  • In a for loop with inner closures, the let keyword binds each closure to the current block, without this each closure in the loop will have the same lexical environment
  • Use closures wisely as they are costly with regards to performance

Module Pattern

  • Utilizing closures to ensure privacy of data
  • Module: small unit of reusable, maintainable, non nampespace polluting code

HTTP

  • Network: Pysical linkage of computers that share data
  • Internet: Global network of networks
  • Web: Way of accessing sources over the internet
  • Gateway: Links your computer to the internet
  • Proxy: Used with navigation over the web, can act like a gateway or cache
  • Cache: Stores HTTP responses to quicken HTTP requests
  • Web Server: Serves documents based off of requests
  • TCP: Connection is made and kept alive as long as two ends are still exchanging messages.

Tips & Tricks

Odds & Ends

  • Coercion: The process of converting one type to another
  • ==: check value, coercion is allowed
  • ===: check value & type, no coersion
  • use strict: Throws errors on mistakes, such as declaring a variable without the var keyword within a function
  • null: Object is not available
  • undefined: Object is not initialized
  • Prototype Design Pattern: Creates objects wwith values copied from a prototype
  • Isomorphism: For every character in string A there is a corresponding character in String B that matches the frequency
  • Reduce: takes in function, with values cumulative/currElement, as parameter and executes the given function, accumilating the elements
  • Reveal Module Pattern: Module pattern with certain data/methods returned as an object

Developing through the browser

  • Ctrl + alt + i - opens up the developer tools in the current browser
  • F8 - allows you to step through the debugger in developer tools
  • F5 - refreshes the current page
  • localStorage.clear() - clears all information stored in local storage
  • sessionStorage.clear() - clears all informaiton stored in the current session

Developing with VS code

  • alt + click - useful for modifying multiple portions of a file
  • ctrl + x - cuts a segment
  • ctrl + c - copys a segment
  • ctrl + v - pastes a cut/copied segment

Coding Practices

  • Make code modular, each function serves one purpose. Don't abuse this.
